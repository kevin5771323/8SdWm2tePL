# 幼儿园管理系统 java1133

## 项目概述

幼儿园管理系统是一款前后端分离的一站式解决方案，旨在助力高效管理，简化幼儿园的日常工作流程。

## 技术栈

- **后端：** Java、SpringBoot、Mybatis
- **前端：** Vue
- **数据库：** Mysql
- **开发工具：** IDEA、JDK1.8、Maven

## 功能模块

### 系统角色

系统设有以下四种用户角色：

- 管理员
- 后勤主任
- 教师
- 家长

### 功能介绍

- **管理员：**
  - 管理职工信息

- **后勤主任：**
  - 处理个人信息
  - 审批职工请假
  - 汇总考勤
  - 制定食谱
  - 管理缺失物品

- **教师：**
  - 管理个人信息、幼儿信息
  - 审批请假
  - 安排课程与活动
  - 打卡考勤
  - 提交缺失物品信息
  - 批复特殊情况
  - 管理班级与体检信息

- **家长：**
  - 管理个人信息
  - 提交幼儿信息与请假审批
  - 查看教师、食谱、体检信息
